API connection guide
1 — Meta Business API (Instagram + Facebook)
Go to developers.facebook.com → Create App → Add "Instagram Graph API" → Connect your Business account. Generate a long-lived Page Access Token. In your backend .env file, set META_ACCESS_TOKEN=... and META_PAGE_ID=....
2 — Google Analytics 4
Go to console.cloud.google.com → Create project → Enable "Google Analytics Data API" → Create a Service Account → Download the JSON key. Share your GA4 property with the service account email (Viewer role).
3 — TikTok
Export weekly from Creator Center → Analytics → Export data and upload via the Weekly Sync panel.
4 — Sporty
Export registrations from Admin → Reports → Registrations → Export CSV and upload via the Weekly Sync panel.
